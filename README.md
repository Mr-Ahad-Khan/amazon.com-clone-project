# Amazon Homepage Clone 🛒

This project is a **front-end clone of the Amazon homepage** built using **HTML and CSS**.
The goal of this project was to practice **layout design, CSS styling, and building a structured web interface similar to a real-world website**.

The page recreates key sections of Amazon’s homepage such as the **navigation bar, product sections, and overall layout**, focusing on improving front-end development skills.

---

## 🚀 Features

* Amazon-style **navigation bar**
* Product category sections
* Clean and structured layout
* Styled using pure CSS
* Beginner-friendly front-end project

---

## 🛠️ Technologies Used

* **HTML5** – Structure of the webpage
* **CSS3** – Styling and layout design
